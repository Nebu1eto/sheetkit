# SheetKit API 레퍼런스

SheetKit는 Excel (.xlsx) 파일을 읽고 쓰는 Rust 라이브러리이며, napi-rs를 통한 Node.js 바인딩을 제공합니다. 이 문서는 Rust 크레이트와 TypeScript/Node.js 패키지에서 사용 가능한 모든 공개 API 메서드를 다룹니다.

---

## 목차

- [워크북 입출력](./workbook.md) - 워크북 생성, 열기, 저장; 워크북 형식 프로파일링, VBA 보존, 확장자 기반 저장
- [셀 조작](./cell.md) - 셀 값 가져오기 및 설정, 셀 값 타입
- [시트 관리](./sheet.md) - 시트 생성, 삭제, 이름 변경, 복사
- [행과 열 조작](./row-column.md) - 행과 열 삽입, 삭제, 복제; 높이, 너비, 표시 여부 및 아웃라인 레벨 관리
- [스타일](./style.md) - 글꼴, 채우기, 테두리, 정렬, 숫자 형식, 보호 스타일; 스타일 빌더 및 중복 제거
- [차트](./chart.md) - 차트 생성 및 관리 (43가지 차트 유형)
- [이미지](./image.md) - 이미지 삽입 및 관리
- [데이터 기능](./data-features.md) - 셀 병합, 하이퍼링크, 데이터 유효성 검사, 코멘트, 자동 필터, 조건부 서식, 테이블, 데이터 변환 유틸리티
- [고급](./advanced.md) - 틀 고정/분할, 페이지 레이아웃, 정의된 이름, 문서 속성, 워크북 및 시트 보호, 수식 평가, 피벗 테이블, 스트림 라이터, 유틸리티, 스파크라인, 테마 색상, 서식 있는 텍스트, 파일 암호화, 시트 보기 옵션, 시트 표시 여부

## API 개요

이 레퍼런스의 모든 섹션에는 **Rust**와 **TypeScript/Node.js** 코드 예제가 포함되어 있습니다. 탭 또는 코드 블록 헤더를 따라 사용 중인 언어에 맞는 구현을 찾으세요.

더 간단한 소개를 원하면 [사용자 가이드](../guide/index.md)를 참조하세요.
